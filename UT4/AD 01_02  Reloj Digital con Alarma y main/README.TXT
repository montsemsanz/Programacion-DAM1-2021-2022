Proyecto: Visorreloj
Authors: David Barnes and Michael Kolling

Capítulo 3

Para utilizar este proyecto crear una instancia de la clase VisorReloj. 
Inspeccionar esta instancia y dejar la ventana del inspector abierta.
LLamar entonces a los métodos del objeto como emitirTicReloj u setHora. Ver la cadena "visorCadena" en el inspector.
El atributo visorCadena simula el actual dispositivo visor de reloj. 

Este proyecto no implementa adecuadamente el manejo de errores.
